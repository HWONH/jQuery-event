클릭 이벤트

클릭 이벤트간의 주의사항 - href 속성에 의한 페이지 이동을 막아야 하는 경우

1. 스크립트 코드 내부에서 막는 방법1 - 클릭 이벤트를 종료하기 직전에 return false;를 작성한다.

2. 스크립트 코드 내부에서 막는 방법2 - 클릭 이벤트 내부에서 실행문들이 시작하기 직전에 매개변수(e)를 설정한 수 e:preventDefault(); => 초기화를 막겠다는 의미로, 새로고침 현상이 발생하지 않는다.

3. href의 속성을 제거한다. href="javascript:void(0);" => 자바스크립트에서 제공되는 모든 값을 아무것도 주지 않겠다는 의미

마우스 관련 이벤트 : mouseover, mouseout, hover(mouseenter, mouseleave)
